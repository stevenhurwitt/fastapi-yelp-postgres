üçé Installation Steps for macOS (Safari/Chrome/Edge)

  1. Download the certificate using the button above
  2. Open Keychain Access:
    • Press Cmd + Space and type "Keychain Access"
    • Or go to Applications ‚Üí Utilities ‚Üí Keychain Access
  3. Import Certificate:
    • Select "System" keychain in the left sidebar
    • Drag and drop the certificate file into Keychain Access
    • Or use File ‚Üí Import Items...
  4. Trust the Certificate:
    • Double-click the "192.168.0.9" certificate
    • Expand the "Trust" section
    • Change "When using this certificate" to "Always Trust"
    • Enter your Mac password when prompted
  5. Restart your browser
  6. Visit https://192.168.0.9 - should show secure lock üîí
